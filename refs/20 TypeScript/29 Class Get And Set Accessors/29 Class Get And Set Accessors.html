TypeScript [29 Class Get And Set Accessors ] Main Topics :

Main Subjects in this lesson will be => practical examples on Class getter and setters [get() | set()] methods and access to [class different modifiers] in the [TSOOP] :

  1. Class getter and setters [get() || set()] methods concept in the [TS] :
    • Using class getter [get()] and class setter [set()] with TSOOP to get access and modify Class properties defined by different types, speially properties definedy by {private | protected} Modifires , which can NOT be accessed by using default methods
      • /
    • [get()] method main features in the [TS] :
      • {concept} => a class method to get a certain class defined property
      • {usages} => used to get a certain defined class properties or more
      • {properties} => defined within the Class and outside the constructor function
      • {syntax} =>
        get getMethodName() : datatype {
        return this._propName ;
        }
    • [set()] method main features in the [TS] :
      • {concept} => a class method to get a certain class defined property
      • {usages} => used to get a certain defined class properties or more
      • {properties} => defined within the Class and outside the constructor function
      • {syntax} =>
        set setMethodName(value:datatype) {
        this._propName = value ;
        }
    • Vip Notes on using [get() || set()] methods in the [TS] :
      • [Get and Set] methods are used with properiteis defined with {private | protected} class Modifires
      • Must define the get or set properties by usign the {_} , to be special and unique for the TS compilier , as privaate or protected class properties
    • Syntax Examples :
      // A] Define main {Class} with using the Class Get() and set() methods + Inner Modifiers [public | private | protected] propeties :
      class className {
      // Assignment of the {Class Modifiers} & {Type Annotation} of Class's properties :
      funcProp : () => datatype4 ;
      // Constructor function with inner Class Modifires [constructor function's parameters] :
      constructor (private _param1 : datatype1 , protected _param2 : datatype2, public param3 : datatype3 ) {
      // Allocating the defined properties to the constructor parameters :
      this.funcProp = function () {
      return `
      prop1 is ${this.param1}
      prop2 is ${this.param2}
      prop3 is ${this.param3}
      ` ;
      }
      }

      // Define a public { get() } method to get the [private property] value :
      get param1():datatype1 {
      return this._prop1
      }

      // Define a public { set() } method to set the [private property] value (with the parmaters' value) :
      set param1(value:datatype1) {
      this._param1 = value ;
      }

      // Define a public general method {getMthd()} :
      genMthd() {
      return `Hello this is a general public method` ;
      }

      }

      // B] Define extracted object from the Defined Class :
      let obj = new className(prop1Value , prop2Value ) ;


      // C] Get the {private} property value by using [get()] method :
      console.log(obj.param1) ;


      // D] Set the {private} property value by using [set()] method :
      obj.param1 = 'newValue' ;


      // E] Get the new {private} property value by using [get()] method [after modification] :
      console.log(obj.param1) ;


      // F] Printing and accessing the defined class properties from the extracted object [by using default method] :
      console.log( `
      The first property [prop1] is ${obj._param1} => [will display an Error due the property is {private} and can NOT be accessed from outside the Class]
      The first property [prop1] is ${obj._param2} => [will display an Error due the property is {private} and can NOT be accessed from outside the Class]
      The third property [prop3] is ${obj._param3} => [will display the property value , due the property is {public} ]
      The functional property [funcProp] is ${obj.funcProp() } => [will through an Error if called from the outside the class , because a private and protected properties ]
      `) ;

      // G] Printing and general method :
      console.log (genMthd()) ;

  • using the [Class Getters and Setters ] methods with defied properties within in Class [TS OOP] requires -> to be defined using {_} before the property name with Modifires private || protected
