JS OOP [Constructor Function Dealing With Properties] Concept:

=> JS OOP [Constructor Function Dealing with Properties]: This lessson will provide to developer more enhanced tools and script to control the constructor function's Properties within the each [Extacted Objects] + and [Pattern Constructor Fucntion] .

=> JS OOP [Constructor Function Dealing with Properties new keywords]: There are (2) Methods to deal with properties in Objects as following :

  1. {Dealing with properties For each extracted Object }; = > this is the default method to deal with properties single for each Extracted Object [Adding new property directly in each Object]
  2. {Dealing with properties as Parametres in Contructor Function [Pattern]}; = > this is the advanded method to deal with properties Globally to All of Objects which will be Extracted later from this Pattern [Constructor Function]
  3. {Merging between Array and Object(s)}; = > This advanded method to deal with Multiple Objects as Array's Elements to get new extracted Objects (which has been Established from the Constructor function) as Array Elements.
