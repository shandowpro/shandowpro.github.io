Main Concept of Function Advanced Self Invoke :

=> [JS Function Advanced Self Invoke] is a way to Code the Function and Invoke it without a need to write the Invoke (Calling) Code or Create an Element Event on page to Invoke the function .

=> [JS Function Advanced Self Invoke] will be Invoked automatically by opeining the page by itself.

Basic Function Properties:

=> [JS Basic Function ] Properties can be as following :

  1. [Basic Functions Main Types] :
    1. [Built-In Functions] =>   They are Functions Built in the main Javascript Syntax Code , and it has certain Programatical task(s), For Example : [Math.random(); => to write a random Number].
    2. [Defiened Functions] =>   They are Customized Functions which built by the developer to implement specefic Code(s) or Procedure(s).
  2. [Basic Functions Syntax Rules] :
    1. Javascript is [ Irrelevant Programing Language ] which is meaning that (Codes Syntax Should Not be according to a Logic arrangement)
    2. The Following Example will be written in (2) Methods (Relavent & Irrelavent) and the [both are Correct in Javascript]:

      3. [Method 1 (Defualt Logic/Relvant Method)] => [Defining the Function First then Invoke it] :

      funtction myFun(parameters ) { //[Function May Not has any Parameters]
          Funtction's Codes & procedures.
          return ; //[Function May Not has Return]
      }

      myFun(); //[This is the Default Invoke to the Function to Process]

      [Method 2 (Defualt Not Logic/Irrelavent Method)] => [Invoke the Function First then Defining it]:

      myFun(); //[This is the Default Invoke to the Function to Process]

      funtction myFun(parameters) { //[Function May Not has any Parameters]
          Funtction's Codes & procedures.
          return ; //[Function May Not has Return]
      }

Javascript [ Function Advanced Self Invoke ] Usages & Benefits :

=> [ Function Advanced Self Invoke ] Properties can be as following :

  1. It very usful specially in advanced programaticaly Auto Calculation processes (self activated Functions )
  2. It can be invoked or implemented just when the page loaded, which is very helpful in some functions in applications which depending on just a {API} Address or User Country which the browser get when just the page is loaded.
  3. It can be used as a programable reactions to certain events (such as inserting input in specific fields in webpage Which will be self invoked just when any or certain input is inserted ).

Function Advanced Self Invoke Syntax :

=> Function Self Invoked Syntax can be by (2) ways as follwing:

  1. First way:
    2. (function myfun() {
    Function Codes & procedures ...
    }());
  2. Second way:
    3. (function myfun() {
    Function Codes & procedures ...
    })();
