HTML5 Lesson 23 Form Input Attributes [Placeholder, Multiple, Required]

  1. Form =>   < input    placeholder=""   >

Form => Input Attribute [ < input   placeholder=""   > ]

Input Attribute [Placeholder] Concept:

=> Input [Placeholder Attribute] is one of the old HTML5 Form's Input Attributes.

=> Input [Placeholder Attribute] is to make a temporary illustrative Text for the user inside the input feild to guide/help him what to insert into this input feild.

=> The Temporary Text displayed in the input feild with this Attribute, will NOT be sent to the main form caption.

=> Input [Placeholder Attribute] requiring a Textual input feild type.

=> Input [Placeholder Attribute] can be allocated for more than one Input Tag outside the form with the page.

Main & Features of Input Attribute [Placeholder Attribute] Usages:

=> It mainly designed to help user in inserting the correct valid content

=> This Attribute can compensate adding more Labels inside the form, and adding guiding placeholder attribute instead.

Example:

Filling multiple Form Inputs Fields with placeholder attribute.

Filling Main Form Inputs Fields guiding the explanatory text inside [Placeholder] attribute:



<section>
   <h3> Filling Main Form Inputs Fields guiding the explanatory text
     inside: </h3>
   <form   action="process.php"   method="get"   >

     <input   type="text"  name="username1"   placeholder = "Enter your
       Username"  >

     <input   type="password"  name="password"   placeholder = "Enter your
       Password"  >

     <input   type="sumbit"  value="send Default" >
   </form>

</section>
Form => Input Attribute [ < input   multiple   > ]

Input Attribute [Multiple] Concept:

=> Input [Multiple Attribute] is one of the old HTML5 Form's Input Attributes.

=> Input [Multiple Attribute] is to make multiple content inserting ability for this input feild (with the same type ofcourse).

=> Input [Multiple Attribute] applicable for any input feild type (except submit & button).

=> Input [Multiple Attribute] can be allocated for more than one Input Tag outside the form with the page.

Main & Features of Input Attribute [Multiple Attribute] Usages:

=> It mainly designed to allow inserting multiple contents with input feild type.

=> This Attribute is very helpful specially when inserting multiple files in Input of type (file) & (Email) commonly.

Example:

Inserting multiple inputs in the same Input Feild using [Multiple] Attribute.

Fulfill your data into following input feilds with [Multiple] advantage :



<section>
   <h3> Fulfill your data into following input feilds with [Multiple]
     advantage : </h3>
   <form   action="process.php"   method = "get"   >

     <input   type="email"  name="mail"   placeholder = "Enter your
       Email(s)"  multiple >

     <input   type="file"  name="uploading"   placeholder = "Enter your
       File(s)"  multiple >

     <input   type="sumbit"  value="send Default" >
   </form>

</section>
Form => Input Attribute [ < input   required   > ]

Input Attribute [Required] Concept:

=> Input [Required Attribute] is one of the old HTML5 Form's Input Attributes.

=> Input [Required Attribute] is to set the current input feild Required to insert a content into it (Obligatory Feild).

=> Input [Required Attribute] applicable to any kind of input feild (Except Submit & Button).

=> Input [Required Attribute] can be allocated for more than one Input Tag outside the form with the page.

Main & Features of Input Attribute [Required Attribute] Usages:

=> It mainly designed to set specific Input Feild(s) Obligatory and preveting the form to Submit other Input feild(s) with out it / them.

Example:

Fulfilling all required Input Feilds in the Form.

Insert all Required Input Feilds in the following Form:




<section>
   <h3> Insert all Required Input Feilds in the following Form: </h3>
   <form   action="process.php"   method = "get">

     <input   type="text"  name="username1"   placeholder = "Enter your
       Username (Required)"  required >

     <input   type="password"  name="password1"   placeholder = "Enter your
       Password (Required)"  required >

     <input   type="sumbit"  value="send Default" >
   </form>

</section>
