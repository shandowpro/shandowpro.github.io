Bootstrap3 Design1 [10 Twitter Bootstrap 3 Design 1 - Grid System Column Nesting] Main Topics :

> Main Subjects {Classes} in this lesson are as following :

  1. {img-responsive} => To make image inside the it's parent responsive relativly with the it's parent container width of Grid
  2. {center-block} => To make image/element in the center of the screen [only in Small & Extra Small]
  3. {lead} => To format the Paragraph or Text with beautiiful Formating and Alignment

Bootstrap3 Design1 [10 Twitter Bootstrap 3 Design 1 - Grid System Column Nesting] Concept:

=> [ 10 Twitter Bootstrap 3 Design 1 - Grid System Column Nesting ] :

  1. Column Nesting =>

    This lessonm is one of the most important lesson in reponsive issue

    Nesting is to maintain the Responsivness of the Children elements inside the A Column Container

    To Acheive Nesting => We must put a [row container] inside the Column (which has Responsive Children Element)

    In Responsivness => It is Recommended to put more Columns in A Row => within the larger Screen Scale
    & Put Less Columns in A Row => within Smaller Scale

  2. Image class {img-responsive} => It will make the image responsive with it's parent container even with changing the Parent box size by Grid Width
  3. Paragraph class {lead} => It will make the Paragraph Text Content responsive with it's parent container , and give it better Formating
  4. Image/Element class {center-block} => It will make the Element Alignment in Center and Block => only in {sm + xs} Screen Scales
