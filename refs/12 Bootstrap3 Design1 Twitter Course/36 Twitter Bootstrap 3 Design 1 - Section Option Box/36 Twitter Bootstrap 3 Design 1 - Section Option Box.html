Bootstrap3 Design1 [33 Twitter Bootstrap 3 Design 1 - Using {Google Fonts}] => General Practical Applications :

= > Creating {Option Tool Box} => with [class="option-box"] for the user To controling within this webpage formating options

= > The {Option Tool Box [.option-box] will include the following Formating :

  • Fixed position: fixed => to be fixed to user where ever he scrolling
  • left : 0 => to be just beside left of the page
  • top : 180px => To take a chance for the nav bar to be viewed clrearly
  • display : none;
  • border: none;
  • padding : 10px;
  • z-index : 9999;

= > The {Option Tool Box [.option-box] will include the following components of Inner boxes {Style Tools} => {will be multiple , but will include only one color Tool {.color-box} } ::

  • First [div] Tool {color option} => with Class= {.color-option}, => Will provide tool to controling Template Website color => and including the following components :
    1. Heading of the Tool [h4]
    2. List of the Tool [ul] with class="list-unstyled" => including => [li] -> empty Value

    This div {.color-box} including the following Formating :

    • border: 3px solid #E41B17; => With the same color of the whole Template
    • float : left
    • padding: 10px => To give inner chidren more obvouised view
    • color: #fff => To Child Heading better view
    • background: #333 => Best color of the background-color for better view of the color palets
      • ;
    • width : 200px => Suit the Number of color available
    • display: none => To be controled by clicking on the Gear button => {Q-query}

  • Gear Font icon button => with Class= {gear-option}, Including the following formating :
    1. Rotating Animation [360 deg for 3s]
    2. color:#fff => for better view
    3. cursor: pointer; => to appear as a Button for the user
    4. padding: 10px => For better view with the background color and font color [as a gear]
    5. background: #E41B17 => the same color of the template fo the website
    6. animation: rotate 3s infinite linear => Animation of the Rotating effect
    7. border:none => To zero the bootstrap previous Formating
    8. border-radius: 50%; => Circular shape suits the rotation animation effect
