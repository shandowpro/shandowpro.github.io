Bootstrap3 Design1 [Start Coding] Concept:

=> [ Start Coding ] :

  1. Include addition (2) files to the HTMl Page [meta] =>
    1. { < meta http-equiv="X-UA-Compatible" content="IE=edge" > } :

      For defining the IE browser to modern Script

    2. { < meta name="viewport" content="width=device-width, initial-scale=1" > } :

      For Mobile Full responsive

  2. IE compatiblity (2) JS Files =>
    1. { < script src="js/html5shiv.min.js" > </script > }
    2. { < script src="js/respond.min.js"> </script > }
