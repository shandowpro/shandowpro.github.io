Bootstrap3 Design1 [13 Twitter Bootstrap 3 Design 1 - Carousel Part 1] Concept:

=> [ 13 Twitter Bootstrap 3 Design 1 - Carousel Part 1 ] :

  1. This lesson => will be more applicated on the currrent Web page on Carousel Library [Slider]
  2. main Carousel Slider in Bootstrap as following :
    1. Main Container , Main container of all slider other components :
      1. id="anyName" => Will be called by each of other components of the Slider [Slider Indicators items (Dots)]
      2. class="carousel slide" data-ride="carousel"
      3. data-ride="carousel"
    2. Slider indicators :
      • This is the Container of the Slider Navigator Dots [one of slider Navigtor control tool]
      • Has => class="carousel-indicators"
      • Each dot Child element has :
        1. data-target="#myslider" => used to call the Main slider
        2. data-slide-to="slide index number" => The index of the Slider to call
        3. class="active" => Given to the First Slide (Desired to be first to show)
    3. Slider Inner slides Contents [Wrapper] :
    4. This is container of the Inner Sliders(items) inside the Slider Section
    5. Has => class="carousel-inner" role="listbox"
    6. Each item/slide is consists of Item container that has :
      1. Image [Basic]
      2. Slide Caption => will Displayed at bottom by default
      3. Paragraph [optional]
      4. Header [optional]
    7. Slider Contorller :
      • Basicaly include (2) anchors[Button container] of (2) spans with inner glyphicon of Next(left) and Previous(right) , as Slider buttons
      • Each anchor include :
      • Each has class="left/right carousel-control" href="#main slider id" role="button" data-slide="prev/next"
        • Glyphicon of glyphicon-chevron-right || glyphicon-chevron-left

    3. This lesson => will be more applicated on the currrent Web page on Carousel Library [Slider]
