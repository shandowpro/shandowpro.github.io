Bootstrap3 Design1 [9 Twitter Bootstrap 3 Design 1 - Grid System Column Ordering] Main Topics :

> Main Subjects in this lesson are as following :

  1. {col-screenScale-push-pushValue} => To Push the column to the right by certain value in certain Screen Scale
  2. {col-screenScale-pull-pullValue} => To Pull the column to the right by certain value in certain Screen Scale

Bootstrap3 Design1 [9 Twitter Bootstrap 3 Design 1 - Grid System Column Ordering] Concept:

=> [ 9 Twitter Bootstrap 3 Design 1 - Grid System Column Ordering ] :

  1. Class {col-screenScale-push-pushValue} =>

    Used to push the column postion to right by a certain value by Grid units of 12

    This property is also depends on the screen scales value

    In case of missing Screen Scale , it's value will be Calculated upon the next Smaller Screen Scale value

    This value can be Reset by writing the Zero in it's Property Value , and this Reseting usually used in case of Deacvtivate this property in certian Screen Scale

  2. Class {col-screenScale-pull-pullValue} =>

    Used to push the column postion to left by a certain value by Grid units of 12

    This property is also depends on the screen scales value

    In case of missing Screen Scale , it's value will be Calculated upon the next Smaller Screen Scale value

    This value can be Reset by writing the Zero in it's Property Value , and this Reseting usually used in case of Deacvtivate this property in certian Screen Scale

Bootstrap3 Design1 [9 Twitter Bootstrap 3 Design 1 - Grid System Column Ordering] => General Practical Applications :

= > Designing page with [1] Main Container => (2) Rows of following content Distrubtion :

  • First Row [Has => (2) Columns , All in One Screen scale {lg}, as following :
    • Display :
      {Large Screen}: First Col will be pushed to the Right by (8) || Second Col will be pulled to the Left by (4)
      {Other Screens}: As Default ordering
    • Components :
      [First Column: width:4 => Pushed to the Right by(8)]
      [Second Column: width:8 => Pulled to the Left by(4)]
  • Second Row [Has => (3) Columns , In Multiple Screen scales {lg, md(0) , sm} , as following :
    • Display :
      {Large, small , Extra Small Screens}: First Col will be pushed to the Right by (8) || Second Col will be pulled to the Left by (4) || Third col will be pulled to the left by (4)
      {Medium Screen}: As Default ordering
    • Components :
      [First Column: width:4 => Pushed to the Right by(8)]
      [Second Column: width:4 => Pulled to the Left by(4)]
      [Third Column: width:4 => Pulled to the Left by(4)]


First Row include following :
  • [First Column: width:4 => Pushed to the Right by(8)]
  • [Second Column: width:8 => Pulled to the Left by(4)]
First div [width:4 || Pushed to the right by (8) -> in [lg] only] {First Row}
Second div [Width:8 || Pulled to the Left by (4) -> in [lg] only] {First Row}


Second Row include following :
  • [First Column: width:4 => Pushed to the Right by(8)]
  • [Second Column: width:4 => Pulled to the Left by(4)]
  • [Third Column: width:4 => Pulled to the Left by(4)]
First div [width:4 || Pushed to the right by (8) -> in [lg,sm,xs] only {not md} ] {Second Row}
Second div [Width:4 || Pulled to the Left by (4) -> in [lg,sm,xs] only {not md} ] {Second Row}
Third div [Width:4 || Pulled to the Left by (4) -> in [lg,sm,xs] only {not md} ] {Second Row}
