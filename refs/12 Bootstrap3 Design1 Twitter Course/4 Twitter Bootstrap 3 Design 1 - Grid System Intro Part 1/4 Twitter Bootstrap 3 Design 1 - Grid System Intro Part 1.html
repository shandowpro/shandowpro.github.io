Bootstrap3 Design1 [4 Twitter Bootstrap 3 Design 1 - Grid System Intro Part 1] Concept:

=> [ 4 Twitter Bootstrap 3 Design 1 - Grid System Intro Part 1 ] :

  1. Main Concept of the {Grid system} =>

    A Bootstrap System to distribute the page into a certain Grids of Rows and columns

    Grid system seprate the page mainly into Rows, each Row seperated at Max into [12]

  2. Sizes of the Screen in Bootstrap Grid System =>
    1. Large Screen => {lg}
    2. Meduim Screen => {md}
    3. Small Screen => {sm}
    4. Very Small Screen => {xs}
  3. Main Elements Classification in the Bootstrap :
    1. Main Containers :

      The main container of the Web page, and has mainly (2) types, as following :

      • A) Basic fixed Container :

        Width = 1170 px

        Center page

        Value Margins

      • B) Fluid Container :

        Width = Full Screen

        Auto Margins

    2. Row :

      The sub parent Container within the main Container, and has a column Boxes

      Row is the Horizontal box , which including children Columns boxes, with following Sizes :

      Each Column can has various Size and specified Screen scale with following Syntax :

      • Column Sizes => This number represent the Column width as percentage of max column Width (12)
      • Screen Scale Size => The Screen Scale to apply this Width within it

      col-[Screen Size(lg,md,sm,xs)]-[Column width(1->12)], with following examples:

      • Large Screen has (4) Columns in a Row => {col-lg-3} X for each Element
      • Medium Screen has (2) Columns in a Row => {col-md-6} X for each Element
      • Small Screen has (1) Column in a Row => {col-sm-12} X for each Element

      One Column can has multiple size and Screen Scale, with following examples :

      • Large Screen has (4) Columns + Medium Screen has (2) Columns in a Row + Small Screen has (1) Column + Very Small Screen [Mobile] has (2) Column =>
        {col-lg-3 col-md-6 col-sm-12 col-xs-6} X for each Element
