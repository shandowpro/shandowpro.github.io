Bootstrap3 Design1 [Intro & What is Bootstrap3] Concept:

=> [ Intro & What is Bootstrap3 ] :

  1. Download the {Bootstrap3} {v3.7.3} && {jquery} Framework =>

    This framework can be downloaded from the official bootstrap site [getbootstrap.com] {ziped folder}

    After downloading the Bootstrap file => Must be extrcted and included with in each project folder

    Downloaded folder include (3) folders :

    1. css folder => Contain bootstrap CSS files , must include following files :

      {bootstrap.min.css} => Which is the Bootstrap Css Minified [compressed] File => Recommeneded to use if you are not going to upgrade this project bootstrap file version

      {bootstrap.css} => Which is the Bootstrap Css Editable [formated] File

      Can include the style file of my custoimized CSS

      Other files not recommended to be used

    2. js folder => Contain bootstrap JS files , must include following files :

      {bootstrap.min.js} => Which is the Bootstrap Css Minified [compressed] File => Recommeneded to use if you are not going to upgrade this project bootstrap file version

      {bootstrap.js} => Which is the Bootstrap Css Editable [formated] File

      Other files not recommended to be used

      Must include the j-query file with in the page [as script element file] ]

    3. font folder => Contain bootstrap Font Icons [Glyphs Icons] files , will be included by default

  2. Priority of the Bootstrap files and J-query, as follwoning :
    1. First => Bootstrap CSS file -> in Head section by link element
    2. Second => My Customized CSS file -> in Head section by link element [To be able to override the code of the Bootstrap Css ]
    3. Third => J-Query JS file -> Before last of Body by Script element [To be able to run the bootstrap file]
    4. Fourth => Bootstrap JS file -> At last of Body by Script element
  3. Best practise to Include/Adding the CSS Files + JS Files of both {J-Query} & {Bootstrap}] with in the HTML page will be included below
