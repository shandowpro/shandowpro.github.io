Bootstrap3 Design1 [6 Twitter Bootstrap 3 Design 1 - Grid System Hidden _ Visible] Main Topics :

> Main Subjects in this lesson are as following :

  1. {hidden-screenScale} => To Hide a certain Element in a certain Screen Scale
  2. {visible-screenScale-diplayMode } => To Appear a certain Element in a certain Screen Scale

Bootstrap3 Design1 [6 Twitter Bootstrap 3 Design 1 - Grid System Hidden _ Visible] Concept:

=> [ 6 Twitter Bootstrap 3 Design 1 - Grid System Hidden _ Visible ] :

  1. Class {hidden} =>

    To Hide A certain Element in certain Screen Scale ONLY

    This Class include the screen scale Type [lg , md , sm , xs] just after hidden word

  2. Class {visible} =>

    To Appear A certain Element in certain Screen Scale ONLY

    This Class include the screen scale Type [lg , md , sm , xs] just after visible word + Display type


Bootstrap3 Design1 [6 Twitter Bootstrap 3 Design 1 - Grid System Hidden _ Visible] Importance & Benefits :

Bootstrap3 Design1 [6 Twitter Bootstrap 3 Design 1 - Grid System Hidden _ Visible] gives us the Following Features :

  • Advanced abilty to control the visibility / Hidden property of an Element && Connect it with the screen Scale type
  • Can apply this property in only a certain screen scale
