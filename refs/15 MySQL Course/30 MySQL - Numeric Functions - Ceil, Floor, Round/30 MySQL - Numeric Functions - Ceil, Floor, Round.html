30 MySQL - Numeric Functions - Ceil, Floor, Round Concept:

This lesson => Using Numeric functions part(1) with advanced examples in Mysql

Numeric functions has very important usages , through dealing with Numeric values in MYSQL DB Applications

{Rounding} Numeric functions are dealing with Formating Numbers within the numeric columns records in the DB

{Rounding} Numeric functions are as following :->

  1. [Ceil(Numric ColName)] => To Apply Lower Rounding to the this number
  2. [Floor(Numric ColName)] => To Apply Lower Rounding to the this number
  3. [Round(Numric ColName , RoundDecimalNum)] => To Apply the Default Rounding / Custom Rounding to the this number


Pratical Example on [{ceil()} & {floor()} & {round()} ] Numbers functions -> {(1) Table {nums} -> Column {numDbl} ] -> from DB [string] :

  1. [Ceil Rounding function by method: => {ceil(numericColName)}

    1 ) Displaying the {MainNum , ceil(MainNum)} result:


    { select num AS mainNum , CEIL(num) as Ceiled Number from nums ; }

    2 ) Displaying the {MainNum , floor(MainNum)} result:


    { select num AS mainNum , FLOOR(num) as Floored Number from nums ; }

    3 ) Displaying the {MainNum , ROUND(MainNum) || {ROUND(MainNum , RoundingNum)} result:


    { select num AS mainNum , ROUND(num) as DEfRounded Number from nums ; }
    { select num AS mainNum , ROUND(num , RoundDecNumber-1-) as CustRoundedNum from nums ; }
    { select num AS mainNum , ROUND(num , RoundDecNumber-2-) as CustRoundedNum from nums ; }
    { select num AS mainNum , ROUND(num , RoundDecNumber-3-) as CustRoundedNum from nums ; }



Important Notes :

  • [Ceil] Rounding funciton => Basic Rounding to the UPPER number , and can be usd in some application
  • [Floor] Rounding funciton => Basic Rounding to the LOWER number , and can be usd in some application
  • [Round] Rounding funciton => The Default / Customized rounding , is the most accurate mathimatical Rounding numeric function
