06 jQuery - Effects - SlideDown _ SlideUp _ SlideToggle Concept:

This lesson => The most common Effects [Sliding] in jquery [syntax and practical applications ] web page as following :

  • Effects preffered included with in the {ready event} => To garunttee the loading of all elements in the web page
  • Effects are built-in functions in the JQ that followed by a function or more , and can be implemented for a certain element [the same element or another]
  • Effects can be nested code for the same element or child(s) inside the main element or a nother elements
  • {Slide} : to controling each [height] & [display] element property , and it has the several types as following :
    1. SlideDown(); => To slide hidden Element down until to be displayed
    2. SlideUp(); => To slide displayed Element down until to be hidden
    3. SlideToggle(); => To switching between hidden/dispalyed Element until to be sliding down/up
  • {Sliding} : Effects Has (2) inner flags or options for each effect , as following :
    2. SlideDown(); || SlideUp(); || SlideToggle(); => has main ( 2 ) flags/options :
      • [Speed value] => can be as one of the following :
        1. Default value (empty field) -> [400 ms]
        2. 'slow'
        3. 'fast'
        4. numberic value -> customized value
      • [Call back function(s)] => Could be customized function or another built in function, Also can hold endless number of nested call back funciton


Pratical Example on [ practical application on the jquery Effects functions ] -> :

  1. 1) [Effect { slideDown() } ]

    {Assigning Basic simple structure of {slideDown)} effect function inside the button [click event] to sliding down another displayed element {p} } :

    $(document).ready(function() {
    $('button').click() {
    function(){
    $('p').slideDown('fast/slow/value' );
    }
    );
    });

    {Assigning Advanced [ flags -> speed & call back function(anoher Sliding down for the same parent element -after hiding the child element ) ] structure of {slideDown()} effect function inside the button [click event] to SlideDown displayed another element {p} } :

    $(document).ready(function() {
    $('button').click() {
    function() {
    $('p').slideDown('fast/slow/value' , function() {
    $(this).slideDown(fast/slow/value);
    });
    }
    );
    });





  2. 2) [Effect { slideUp() } ]

    {Assigning Basic simple structure of {SlideUp()} effect function inside the button [click event] to Sliding Up another element {p} } :

    $(document).ready(function() {
    $('button').click() {
    function(){
    $('p').slideUp('fast/slow/value');
    }
    );
    });

    {Assigning Advanced [ flags -> speed & call back function(anoher hide for the same parent element ) ] structure of {SlideUp()} effect function inside the button [click event] to hide another element {p} } :

    $(document).ready(function() {
    $('button').click() {
    function() {
    $('p').SlideUp('fast/slow/value' , function() {
    $(this).SlideUp('fast/slow/value');
    });
    }
    );
    });



    {Assigning one call back function including (double funcitons ) to be exected at once -> by click event:

    $(document).ready(function() {
    $('button').click() {
    function() {
    $('p').slideUp('fast/slow/value', function() {
    $('span').slideUp('fast/slow'/value);
    $(this).slideUp('fast/slow'/value);
    });
    }
    );
    });





  3. 4) [Effect { slideToggle() }]

    {Assigning Basic simple structure of {slideToggle()} effect - with speed & call back function - inside the button [click event] to SlideToggle another hidden element {p1} } &&
    {Assigning Basic simple structure of {slideToggle()} as a call back function inside the main effect function of {p} , to slideToggle a the same button of event click {this} }

    $(document).ready(function() {
    $('button').click() {
    function(){
    $('p').slideToggle('fast/slow/value' , function() {
    $(this).slideToggle('fast/slow'/value');
    });
    }
    );
    });
