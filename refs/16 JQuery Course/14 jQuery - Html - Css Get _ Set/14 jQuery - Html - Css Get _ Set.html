14 jQuery - Html - Css Get _ Set Concept:

This lesson => The main Css functions dealing with Classes in jquery [syntax and practical applications ] web page as following :

  • elem.css("CssPropName"); => Get Element CSS Properties
  • elem.css("CssPropName", "CssPropValue"); => Set Element single CSS Property
  • elem.css({"CssPropName1":"CssPropValue1" , "CssPropName2":"CssPropValue2" , "CssPropName3":"CssPropValue3" }); => Set Element multiple CSS Properties


Get & Set element(s) CSS properties , can be with one or more CSS properties

Get & Set element(s) CSS properties , can be used with advanced by several methods , such as :

  • Set a certain element's css property with anothner element's val()
  • Get a certain element's css property as another element's val()
  • Set a new element with Css properties & append it with in the web page

Pratical Example on [ practical application on the jquery HTML elements Get & Set CSS properties methods ] -> :

  1. A) [Simple Set] {elem.css();} -> function : [Creating new element & Set it by several CSS properties & Append the element in the webpage]

    1) Creating new element [When page is being loaded] :

    $(document).ready(function() {
    var elem = '<div></div> ';
    });



    2) Appending the new created element into the webpage [before set the CSS properties ] :

    $(document).ready(function() {
    $('body').append(elem);
    });


    3) Set the new element CSS properties [single CSS property] :

    $('div').css("color" , "#f00");

    4) Set the new element CSS properties [multiple CSS properties] :

    $('div').css({
    "color":"#f00",
    "border":"3px solid red",
    "fontSize":"26px",
    "backgroundColor":"#EEE"
    });

    5) Advanced creating,pending, adding css properites[by another css property value] of element(s)

    $('document').ready(function(){
    1) Defigning css properties values :
    var color = $('p').css('color'),
    var fontSize = $('p').css('fontSize'),
    var border = $('p').css('border');
    2) Adding css properties to the element & append it to the parent :
    $('<span></span>'.{
    "color": color,
    "fontSize": fontSize ,
    "border": border ,
    }).appendTo('body');
    });


  2. B) [Advanced Set] {elem.css();} -> function : [Set an element's CSS property with another element's [input] val

    Execute the function [When click button] :

    $(document).ready(function() {
    $('button').click(function() {
    var elem = $('div'),
    input1 = $("input1"),
    input2 = $("input2"),
    input3 = $("input3");

    // a) Set [single CSS property {color}] :
    elem.css("color" , input1.val());

    // b) Set [multiple CSS properties {color , fontSize , background} WITH OTHER ELEMENT'S VALUE AND SIMPLE VALUES ]:
    elem.css({
    "color" : input1.val(),
    "fontSize": input2.val(),
    "background": input3.val(),
    "border": "3px solid #f00"
    }) ;
    });





  3. C) [Advanced Get] {elem.css();} -> function : [Get an element's CSS property AS another element's [input] val || [span] text

    Execute the function [When click button] :

    $(document).ready(function() {
    $('button').click(function() {
    var elem = $('div'),
    elem1 = $("input"),
    elem2 = $("span");
    // a) Get [Single Element's Single CSS property {color}] -> as a [the same element] text :
    elem1.text( this.css("color") );

    // b) Get [Single Element's Multiple CSS properties {color , fontSize , background} -> as a [input] value && [span] text :
    elem1.val( elem.css("color") );
    elem2.text( elem.css("fontSize") );

    }) ;
    });
