23 jQuery - Selectors Reference Part 4 Concept:

This lesson => The main Selector Refernces functions {part 4} in the JQ functions in jquery [syntax and practical applications] web page as following :

  • Main Selector Refernces [part 4]:
  • Main functions :
    1. 1) [Specific elements tag type] selectors group :
      • [ $(':header') ] =>
        1. To select All elements with type of head {from h1 to h6}
        2. Can be combined with other types of selector
        3. Can be used for several elements with in the same page
        4. Note: In case of using this selector inside a container -> This selector require > {Sub container of header it will include || * if it in the first level}


    2. 2) [Has a certain child tagname] selectors group :
      • [ $("elem:has('tagName/className/TagName')") ] =>
        1. To select All elements that has certain child type
        2. Can be combined with other types of selector
        3. Can be used for several elements with in the same page


    3. 3) [specific events] selectors group :
      • [ $(":animated") ] =>
        1. To select any element that has an animattion funciton on it
        2. Can be combined with other types of selector
        3. Can be used for several eslements with in the same page

      • [ $(":focus") ] =>
        1. To select any element that has focus effect on it
        2. Can be combined with other types of selector
        3. Can be used for several elements with in the same page


    4. 4) [Empty & Has contents] selectors group :
      • [ $("elem:contains('content')") ] =>
        1. To select a certain element that include a certain content
        2. Can be combined with other types of selector
        3. Can be used for several eslements with in the same page

      • [ $(":empty") ] =>
        1. To select All elements which has NO any contents [even an empty spaces is a content]
        2. Can be combined with other types of selector
        3. Can be used for several elements with in the same page

      • [ $(":parent") ] =>
        1. To select any element(s) that has any content [is not empty]
        2. Can be combined with other types of selector
        3. Can be used for several elements with in the same page


    5. 5) [Hidden and visibility] Selector :
      • [ $("elem:hidden") ] =>
        1. To select any element of (certian type) which is hidden [which can be {dislay : none} OR {width/height:0 } ]
        2. This selector will not select the element with [visibitlty:hidden ]
        3. Can be combined with other types of selector
        4. Can be used for several elements with in the same page
      • [ $("elem:visible") ] =>
        1. To select any element of (certain type) which is visible and not hidden [but may be not visible ]
        2. This selector will select the element with [visibitlty:hidden]
        3. Can be combined with other types of selector
        4. Can be used for several elements with in the same page

    6. [ $(":root") ;] ->
      • Select the element which is the main parent of the webpage document [which is commonly is {html}]
      • Can be combined with other types of selector
      • Can be used for just one element in a webpage

    7. [ $(":lang('language')") ] ->
      • Select any element is having a certain value of the lang attribute
      • Can be combined with other types of selector
      • Can be used for just one element in a webpage


Pratical Example on [Practical application on the jquery selectors of elements various types ] -> :

  1. A) First group [ Specific elements tag type Selectors] [ 1 functions] {:header}

    1] Using [ {:header} ] to select all types of headers elements in the webpage :

    $(document).ready(function() {
    $(':header').css("color","red"); => {Apply css changes on the all types of headers types in wghoel page }
    });




  2. B) [ Has a certain child {tagname, class , id}] Group => { $("elem:has('tagName')") } ] [1 function] simple change ( css properety ]

    1] Using [ $("elem:has('tagName')") ] method :

    $(document).ready(function() {
    $("div:has('p')").css("color","red"); => {select any Div element that has a child of TagName type (P)}
    $("div:has('.className')").css("color","red"); => {select any Div element that has a child of ClassName '.className'}
    $("div:has('#idName')").css("color","red"); => {select any Div element that has a child of idName '#idName'}
    });




  3. C) [with specific events] Group => { $(":animated") & $(":focus") } -> simple change (css properety) ]

    1] Using [ { $(":animated") } ] method to select any elements has animation function :

    $(document).ready(function() {
    $('div:anmiated').css("color","red"); => {select any div has animation function - to apply changes}
    });




    2] Using [ { $(":focus") } ] method to select any input field has focus effect :

    $(document).ready(function() {
    $('input:foucs').css("paddding","20px"); => {select any input has focus effect - to apply changes}
    });




  4. D) [Empty & Has contents & contain a certain contents] Group => { $(":contains") & $(":empty") & $(":parent") } -> simple change ( css properety ]

    1] Using [{ $(":contains('text')") } method to select certain element with certain content :

    $(document).ready(function() {
    $("div:contains('jq')").css("color","red"); => {select the any div that inlcude the text [jq] }
    });




    2] Using [{ $(":empty") } method to select any elements is empty of any contents or even empty spaces :

    $(document).ready(function() {
    $("div:empty").css("color","red"); => {select any empty div element }
    });




    3] Using [{ $(":parent") } method to select any elements is Not empty and has any content of any type even empt spaces :

    $(document).ready(function() {
    $("div:parent").css("color","red"); => {select every div element with it is not empty }
    });




  5. E) [Hidden and visibility] Group {2 functions} => { $(":hidden") & $(":visible") } -> simple change ( css properety: ) ]

    1] Using [{ $(":hidden") } method to select any hidden element with in the page :

    $(document).ready(function() {
    $("div:hidden").fadeIn(1000); => {select any Hidden div and apply the fadeIn effect }
    });




    2] Using [{ $(":visible") } method to select any elements is empty of any contents or even empty spaces :

    $(document).ready(function() {
    $("div:visible").slideToggle(1000); => {select any visible div element and toggling to hidden/show effect }
    });




  6. F) [Solo selectors] Group => { $(":root") & $(":lang('')") } -> simple change ( css properety ]

    1] Using [{ $(":root") } method to select the main element {html} with in the page :

    $(document).ready(function() {
    $(":root").css("color" , "red"); => {select the Html element and applying css effect}
    });




    2] Using [{ $("elem:lang('languageValue')") } method to select any element with attribute with certain value :

    $(document).ready(function() {
    $("div:lang('ar')").css("fontSize" , "50px"); => {select the div element with lang attriblute value 'ar'}
    });
