17 jQuery - Traversing Reference - Is Concept:

This lesson => The main Traversing functions in the JQ functions in jquery [syntax and practical applications ] web page as following :

  • Main HTML Traversing [part 2]:
  • Main functions :
    • [children()] =>
      1. To access the direct children {one level down}
      2. Accept each [normal selectors {tag name , class , id}]-> (providing more accurate access -with one level below-)
      3. Not Accept the advanced filter ,such as {:contains , :first , :last} -> (providing more accurate access -with one level below-)

    • [find()] =>
      1. To access multiple levels children
      2. Accept each [normal selectors {tag name , class , id}] -> (providing more accurate access -with all level below-)
      3. Accept the advanced filter ,such as {:contains , :first , :last}-> (providing more accurate access -with all level below-)


Pratical Example on [ practical application on the jquery HTML elements's children methods ] -> :

  1. A) {elem.children();} -> function : [Access to the direct child of an element]

    I] [Printing the element's all Direct level children ] :

    1) {without the element(s) children selectors } :

    $(document).ready(function() {
    var childElem = $('span').children(),
    Elem = $('span');
    console.log(childElem);
    alert(childElem);
    Elem.text(childElem);
    });




    2) {using certain Element(s) class name} :

    $(document).ready(function() {
    var childElem = $('span').children('.jq'),
    Elem = $('span');
    console.log(childElem);
    alert(childElem);
    childElem.text(childElem);
    });




    II] [Changing single css property of element's direct children ] :

    1) {without using any certain children selectors} :

    $(document).ready(function() {
    var childElem = $('elem').children();
    childElem.css('csspropertyName', 'csspropertyValue');
    });



    2) {using certain element(s) selector:tag name } :

    $(document).ready(function() {
    var childElem = $('elem').children('tagName');
    childElem.css('csspropertyName', 'csspropertyValue');
    });



    3) {using certain element(s) selector:Class name} :

    $(document).ready(function() {
    var childElem = $('elem').children('.className');
    childElem.css('csspropertyName', 'csspropertyValue');
    });



    4) [Changing multiple css properties of element's children] :

    $(document).ready(function() {
    var childElem1 = $('span').children(),
    childElem2 = $('p').children('span'),
    childElem3 = $('div').children('.jq');
    childElem1.css({
    csspropertyName1: 'csspropertyValue1',
    csspropertyName2: 'csspropertyValue2',
    csspropertyName3: 'csspropertyValue3',
    });
    childElem2.css({
    csspropertyName1: 'csspropertyValue1',
    csspropertyName2: 'csspropertyValue2',
    csspropertyName3: 'csspropertyValue3',
    });
    childElem2.css({
    csspropertyName1: 'csspropertyValue1',
    csspropertyName2: 'csspropertyValue2',
    csspropertyName3: 'csspropertyValue3',
    });
    });




  2. B) {elem.find();} -> function : [Access children within certain level(s)]

    I] [Printing the element's all multiple level children ] :

    1) {without using element(s) children selectors } :

    $(document).ready(function() {
    var childElem = $('span').find(),
    Elem = $('span');
    console.log(childElem);
    alert(childElem);
    Elem.text(childElem);
    });




    2) {using certain Element(s) class name} :

    $(document).ready(function() {
    var childElem = $('span').find('.jq'),
    Elem = $('span');
    console.log(childElem);
    alert(childElem);
    Elem.text(childElem);
    });




    II] [Changing single css property of element's direct children ] :

    1) {without using any certain children selectors} :

    $(document).ready(function() {
    var childElem = $('elem').find();
    childElem.css('csspropertyName', 'csspropertyValue');
    });



    2) {using certain element(s) selector:tag name } :

    $(document).ready(function() {
    var childElem = $('elem').find('tagName');
    childElem.css('csspropertyName', 'csspropertyValue');
    });



    3) {using certain element(s) selector:Class name} :

    $(document).ready(function() {
    var childElem = $('elem').find('.className');
    childElem.css('csspropertyName', 'csspropertyValue');
    });



    4) [Changing multiple css properties of element's children multiple levels] :

    $(document).ready(function() {
    var childElem1 = $('span').find(),
    childElem2 = $('p').find('span'),
    childElem3 = $('div').find('.jq');
    childElem1.css({
    csspropertyName1: 'csspropertyValue1',
    csspropertyName2: 'csspropertyValue2',
    csspropertyName3: 'csspropertyValue3',
    });
    childElem2.css({
    csspropertyName1: 'csspropertyValue1',
    csspropertyName2: 'csspropertyValue2',
    csspropertyName3: 'csspropertyValue3',
    });
    childElem2.css({
    csspropertyName1: 'csspropertyValue1',
    csspropertyName2: 'csspropertyValue2',
    csspropertyName3: 'csspropertyValue3',
    });
    });



    5) [Changing single css property of element's children multiple levels - with using advanced filters-] :

    $(document).ready(function() {
    var childElem1 = $('span').find("span:contains('word')"),
    childElem2 = $('p').find('span:first'),
    childElem3 = $('div').find('span:last');

    childElem1.css('CssPropName','CssPropValue');

    childElem2.css('CssPropName','CssPropValue');

    childElem3.css('CssPropName','CssPropValue');

    });
