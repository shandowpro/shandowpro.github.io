26 jQuery - Events Reference - Bind() Concept:

This lesson => The main Events Refernces functions {part 1} -> [bind()] in the JQ functions in jquery [syntax and practical applications] web page as following :

  • Events References functions {part 1} -> [bind()] :
  • Main functions [All events will be added to ONE element] :
    1. First Group : [ Single & Multiple events functions by Default Method [without using {bind()} method ] ] :
      • [ $("elem").event(function(){}); ] -> [Add Single Event to an Element by {default method: without using Bind()} ]

      • [ $("elem").event1(function(){}); || $("elem").event2(function(){}); || $("elem").event3(function(){}); ] -> [Add multiple Events Using to an Element by {default method : without using Bind()} ]


    2. Second Group : [ Single & Multiple events functions by {Bind()} Method (without using {Events Map} ) ] :
      • [ $("elem").bind("eventName", function(){}); ] -> [Add Single Event Using {bind()} Method to an Element by {without using Map method} ]

      • [ $("elem").bind("eventName1 eventName2 eventName3" , function(){} ); ] -> [Add Multiple Events Using {bind()} Method to an Element by {without using Map method} ]


    3. Third Group : [ Single & Multiple events functions by Bind() Method (with using {Events Map} ) ] :
      • [ $("elem").bind(eventName1: function1(){}); ] -> [Add Single Event Using {bind()} Method to an Element by {with using Map method} ]

      • [ $("elem").bind(eventName1: function1(){}, eventName2: function2(){} , eventName3: function3(){} ); ] -> [Add Multiple Events Using {bind()} Method to an Element by {with using Map method} ]


    4. Fourth Group : [ Single & Multiple events functions by Bind() Method (with using {Custom Event + trigger()} - without Event Map , Pars[only Event parameter] ) ] :
      • [ $("elem").bind("eventName1", function1(EventPar){});] & [$("elem").trigger("EventPar"); ] -> [Add Single Event Using {bind()} Method to an Element by {with using Custom method + trigger() + Non Pars} ]

      • [ $("elem").bind("eventName1 eventName2", function1(EventPar){});] & [$("elem").trigger("EventPar"); ] -> [Add Multiple Events Using {bind()} Method to an Element by {with using Custom method + trigger() + Non Pars} ]


    5. Fifth Group : [ Single & Multiple events functions by Bind() Method (with using {Custom Event + trigger() + Event Map} - with/without Pars[only Event parameter] ) ] :
      • [ $("elem").bind("CusteventName1", function(EventPar){}); ] & [$("elem").trigger("EventPar"); ] -> [Add Single Event Using {bind()} Method to an Element by {with using Custom method + trigger() + Event Map + NoPars} ]

      • [ $("elem").bind({CusteventName1: function(EventPar1, par1 , par2 , par3){}, CusteventName2: function(EventPar2, par1 , par2 , par3){} , CusteventName3: function(EventPar3, par1 , par2 , par3){} }] & [$("elem").trigger("EventPar" , ["par1value" , "par2value" , "par3value"]); ] -> [Add Multiple Events Using each {bind()} & trigger() & {event map} & with pars Methods to an Element by {with using Custom method + trigger() + Pars} ]



Pratical Example on [Practical application on the jquery selectors of elements various types ] -> :

  1. First Group : [ Single & Multiple events functions (by Default Method) {without using {bind()} method } ]

    1] Using [ {$("elem").eventName(function(){});} ] to Add Single event {click()} to the element (by default method) :

    $(document).ready(function() {
    $('elem').click(function(){
    $(this).css("color","red"); => {Apply css changes on the element by click on it }
    });


    });




    2] Using [ {$("elem").eventName(function(){});} || {$("elem").eventName(function(){});} ] to Add First and Second events {mouseenter() & mouseleave() } to the element (by default method) :

    $(document).ready(function() {
    Add First Event {mouseenter()} :
    $('elem').mouseenter(function(){
    $(this).css("color","red"); => {Apply css changes on the element by click on it }
    });




    Add Second Event {mouseleave()} :
    $('elem').mouseleave(function(){
    $(this).css("color","blue"); => {Apply css changes on the element by click on it }
    });


    });




  2. Second Group [ Single & Multiple events functions (by Default Method) {with using {bind()} + {without Event Map} method } ]

    1] Using [ $("elem").bind("eventName1" , function(){Codes;}) ] method to Add a Single event to the element [by the bind() method] :

    $(document).ready(function() {
    $("elem").bind("eventName" ,function(){
    $(this).css("color" , "red");
    });
    });




    2] Using [ $("elem").bind("eventName1") ] method to multiple events to the element [by the bind() method] :

    $(document).ready(function() {
    $("elem").bind("eventName1 eventName2" ,function(){
    $(this).css("color" , "red").fadeOut(1000);
    });
    });




  3. Third Group : [ Single & Multiple events functions by Bind() Method (with using {Events Map - without Pars} ) ]

    1] Using [ $(elem).bind({"evetName1": function() {Codes;} }) ] method to Add Single Event by using Bind() + {Event map - without pars} :

    $(document).ready(function() {
    $('elem').bind({
    "eventName1" : function() { $(this).css("color" , "blue").SlideOut(2000); }
    }) ;
    });




    2] Using [ $(elem).bind({ "evetName1": function() {Codes;} , "evetName2": function() {Codes;} }) ] method to Add Multiple Events by using Bind() + {Event map - without pars} :

    $(document).ready(function {
    $('elem').bind({
    "eventName1" : function() { $(this).css("color" , "blue").slideOut(2000); },
    "eventName2" : function() { $(this).slideToggle(3000).css("width" , "150px"); }
    }) ;
    });




  4. Fourth Group : [ Single & Multiple events functions by Bind() Method (with using {Events Map - with Pars} ) ]

    1] Using [ $(elem).bind({ "evetName1": function(par1 , par2 , par3) {Codes;} }) ] method to Add Single Event by using Bind() + {Event map - with pars} :

    $(document).ready(function() {
    $('elem').bind({
    "eventName1" : function(par1 , par2 , par3) {
    $(this).css("color" , "blue").SlideOut(2000);
    $alert('Hello ' + par1 + ' Welcome to ' + par2) ;
    $(this).css("height" , par3 ).SlideOut(2000);
    }
    }) ;
    });




    2] Using [ $(elem).bind({ "evetName1": function() {Codes;} , "evetName2": function() {Codes;} }) ] method to Add Multiple Events by using Bind() + {Event map - with pars} :

    $(document).ready(function {
    $('elem').bind({
    "eventName1" : function(par1 , par2 , par3) {
    $(this).css("width" , "500px").slideIn(2000);
    $alert('Hello ' + par1 + ' Welcome to ' + par2) ;
    $(this).css("width" , par3 ).slideOut(2000);
    },
    "eventName2" : function(par1 , par2 , par3) {
    $(this).css("color" , "blue").fadeOut(2000);
    $alert('Hello ' + par1 + ' Welcome to ' + par2) ;
    $(this).css("color" , par3 ).slideOut(2000);
    }
    }) ;
    });




  5. Fifth Group : [ Single & Multiple events functions by Bind() Method (with using {custom events - {without event Map} & {without Pars} ) ]

    1] Using [ $(elem).bind("customEvent", function(event) {Codes;} }) ] method to Add Single Event by using {Bind() + Custom Event} + {without Event map & without pars} :

    $(document).ready(function() {
    // a) Adding [single] {customEvent} (without Event Map) to the element:
    $('elem').bind("custEvent", function(event){
    $(this).css("color" , "red").fadeIn(1000);
    }) ;

    b) Calling and activating the defined event to the element :
    $('elem').trigger("custEvent");
    });




    2] Using [ $(elem).bind("customEvent1 customEvent2", function(event) {same Codes;} }) ] method to Add Multiple customEvents by using {Bind() + Custom Event (with same codes)} + {without Event map & without pars} :

    $(document).ready(function {
    // a) Adding [Multiple] {2 customEvents (of the same codes)} (without Event Map) to the element:
    $('elem').bind("custEvent1 custEvent2", function(event){
    $(this).css("color" , "red").fadeIn(1000);
    }) ;

    b) Calling and activating the defined custom events to the element :
    $('elem').trigger("custEvent1");
    $('elem').trigger("custEvent2");
    });


    3] Using [ $(elem).bind("customEvent1", function(event) {Codes;} }) ] method to Add Multiple customEvents by using {Bind() + Custom Event (with same codes)} + {without Event map & without pars} :

    $(document).ready(function {
    // a) Adding [Multiple] {2 customEvents (of the same codes)} (without Event Map) to the element:
    $('elem').bind("custEvent1 custEvent2", function(event){
    $(this).css("color" , "red").fadeIn(1000);
    }) ;

    b) Calling and activating the defined custom events to the element :
    $('elem').trigger("custEvent1");
    $('elem').trigger("custEvent2");
    });




  6. Sixth Group : [ Single & Multiple events functions by Bind() Method (with using {custom events + event Map} {with Pars} ) ]

    1] Using [ $(elem).bind({"customEvent": function(event , par1 , par2 , par3) {Codes;} }) ] method to Add Single Event by using {Bind() + Custom Event} + {with Event map & with pars} :

    Each custom event's function contain -> Css changing preoprpties + effect by using parameters :

    $(document).ready(function() {
    // a) Adding [single] {customEvent} (with Event Map + pars) to the element:
    $('elem').bind({
    "custEvent1": function(event , par1 , par2 , par3 ) {
    $(this).css({
    "color" : par1 ,
    "width" : par2 ,
    }).fadeIn(par3) ;
    },
    });

    b) Calling and activating the defined events to the element :
    $('elem').trigger("custEvent1" , ["par1" , "par2" , "par3"]);
    });




    2] [ $(elem).bind({"customEvent1": function(event , par1 , par2) {Codes;} } , {"customEvent2": function(event , par1 , par2) {Codes;} } ) ] method to Add Multiple Events by using {Bind() + Custom Event} + {with Event map & with pars} :

    $(document).ready(function {
    $('elem').bind({
    "custEvent1": function(event , par1 , par2 , par3 ) {
    $(this).css({
    "color" : par1 ,
    "width" : par2 ,
    }).fadeIn(2000) ;
    },
    "custEvent2": function(event , par1 , par2 , par3 ) {
    $(this).css({
    "padding" : par1 ,
    "height" : par2 ,
    }).slideToggle(par3) ;
    },
    });

    b) Calling and activating the defined Custom Events to the element :
    $('elem').trigger("custEvent1" , ["par1" , "par2" , "par3"]);
    $('elem').trigger("custEvent2" , ["par1" , "par2" , "par3"]);
    });


    });
