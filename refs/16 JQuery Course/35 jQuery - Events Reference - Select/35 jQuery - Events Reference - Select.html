35 jQuery - Events Reference - Select concept :

[ select() ] => This method is , as following :

  • Can used to add/assign one or multiple events to the element
  • Also using same parameters of the {on()} => [event name , data , function ]
  • In case of adding multiple events to the same element => assinged function(s) will be processed for each event


The [ select() ] is using usualy in web applications (specially in one proceess needed function)
