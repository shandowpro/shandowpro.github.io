[Api provider] => Get the movie id parameter from the api provider {' movie / {movie.id} '}

  1. 1- Obtain the the API paramater responsible for the {single movie details} => from :
    {Movie details} -> '/movie/{movie_id}' -> Try it -> get the api with the [movie details] as following :
    'https://api.themoviedb.org/3/movie/{movie_id}?api_key=....&langauge=ar' => [this will return the selected movie's details]

  2. 2- Download and setup the ['react-router-dom'] library via the npm by the following code :
    npm i react-router-dom

  3. 3- Import the required classes from ['react-router-dom'] library inside the {App.js} as following :
    import { BrowserRouter, Routes, Route } from "react-router-dom";

  4. 4- Use the imported Routering classes in the return() by wrapping the components and elements desired to be used as Routers , as following :

Vip Notes about routering with using {link} [Requirements-breifly] :

  1. {App.js} =>
    • Must import each of {BrowerRouter , Routes , Route} from the 'react-router-dom' library
    • Wrapped All desired Routes components in this project between [BrowerRouter & Routes]
    • Each Route components => must be wrapped between [Route] , with assigning following props :
      1. [path] => 'the hypernate sub domain of this Route '
      2. [element] => {the full element script}

  2. {Component.js} =>
    • Must import {Link} from the 'react-router-dom' library
    • Wrapped component's script between [Link] , with the following props :
      1. [to] => 'the hypernate sub domain value ' => [typical equal to that used in this Route define in the [path] prop]
      2. [element] => {the full element script}
