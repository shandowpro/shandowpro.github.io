> Main Subjects in this lesson will be => Information about Third ReactJS Project {Movies DB project } [part2] , as following :

Main Subjects :

  1. Main steps of this application :
    1. First => Receiving from the a real live API and assinging it in frontend element :
      • Define the datasource recieved from the real API in a useState variable in {App.js} and send it as a [Prop] into the {MoviesList.js} component
      • Recieving datasource prop in the {MoviesList.js} component , and send it to the {CardMoive.js} as a prop while mapping + inner propreerites of object of datasouce

    2. Second => programming the live search feature :
      • Define the searcing url variable in the [app.js] main component & send it into the NavBar component as a prop

58-Movies DB Project Part 2 concept :

First => Receiving from the a real live API and assinging it in frontend element :

1] {App.js} => Define the datasource recieved from the real API in a useState variable in {App.js} and send it as a [Prop] into the {MoviesList.js} component :

  1. Defining the [useState()] variable to recieve the datasoucrce inside it
  2. Define seperate function of using {axios()} method to recieve the [api] datasource data with using [aync + await] + assiging it inside the useState setter parmanter
  3. Define [useEffect()] method to implement the defined seperate function by using empty parameters [] => to execute ti once by the first rendering
  4. Send the value of useState into the CardList as a prop

    5. // 1] importing section :
    import React,{useState, useEffect} from "react";
    import 'bootstrap/dist/css/bootstrap.min.css';
    import { Container } from "react-bootstrap";
    import NavBar from "./components/NavBar";
    import MoviesList from "./components/MoviesList";
    import axios from "axios";

    // 2] functional component section :
    function App() {
    // a] Define the useState variable :
    const [movies, setMovies] = useState([]) ;

    // b] Define a seperated function of recievcing a api varialbe and assinging it's value inside a useState variable :
    const getAllMovies = async () => {
    // 1- Define variable of recieving the datasource of api [using (axios) method wiht (async await) ] :
    const res = await axios.get('https://api.themoviedb.org/3/movie/popular?api_key=893adba5dca668458f2a181bcbd88e02&language=ar') ;

    // 2- printing testing of the defiend variable of api [accessing to the 'result' prop] :
    console.log(res.data.results);

    // 3- setting [variable of recieving the datasource of {api} as the value of defined [state] [accessing to the 'result' prop] :
    setMovies(res.data.results);
    }

    // c] Define a useEffect to implememnt the denfined sperated function => [to bve caled while the page first rendering] :
    useEffect(() => {
    getAllMovies() ;
    } , [])

    // 3] the return section :
    return (
    < div className="color-body" >
    < NavBar / >
    < Container >
    // Sending the [state] variable of datasource as a prop into the component that using it :
    < MoviesList movies={movies} / >
    < /Container >
    </ div >
    );
    }


    // 3] export section :
    export default App;

2] {MoviesList.js} => Recieving datasource prop in the {MoviesList.js} component , and send it to the {CardMoive.js} as a prop while mapping + inner propreerites of object of datasouce :

  1. Imporing the {CardMoive.js} of the single component
  2. Recieving the sent prop of state variable {movies}
  3. Mapping with prop of state varialbe {movies} inside the Imported the {CardMoive.js} to assign the datasource {movies}
  4. Using condtional rendering wrapper of the Mapping with using the object of {mov} as receved elements [movies ] of the received datasource
  5. Sending the {mov} parameter into the single element {CardMovie.js} as :
    1. mov= {mov} => represent each item while data of datasource object
    2. key= {mov.id} => represent each item's id of datasource object
// 1] importing section :
import React from "react";
import CardMovie from "./CardMovie";
import { Row, Card, Col } from "react-bootstrap";

// 2] functional component section :
const MoviesList = ({movies}) => {
return (
< Row className='mt-3' >

movies.length >=1 ? (movies.map((mov)=> {

}
< /Row >
)
}

// 3] export section :
export default MoviesList;

3] {CardMovie.js} => Recieving datasource prop & inner propreerites in the {CardMovie.js} component to assing each property :

  1. Recieving the sent prop of iner object {mov} as a prop of {CardMovie.js} functional component
  2. Sending the {mov} parameter's inner properties inside each property data , as following :
    1. Movie poster /image -> {`image base url ` + mov.poster_path}
    2. Movie name -> {mov.title}
    3. Movie original name -> {mov.original_title}
    4. Movie release Date -> {mov.release_date}
    5. Movie voter count -> {mov.vote_count}
    6. Movie voter average value -> {mov.vote_average}

Second => programming the live search feature :

1] {App.js} => Define the searching method with parameter of the recieved value of input element , with using conditional if according to the parmenter's value :

  1. Define a function of seraching operation according to the receved parameter value , that include the following :
    • This function contianer a pamreter => [represent the incoming value of input from {NavBar} comopnent ]
    • Conditional if upon the recived paramter value , with the follwing condtions :
      • empty value (=== '') => [means that the input value is empty] => must display the all elements method
      • has a any value => [means the input field has a value] => must start the searching opertion by defining variable of the api search url , with using the paramater value

  2. Passing through the defined variable into the user component as a prop

    3. // 1] importing section :
    import React,{useState, useEffect} from "react";
    import 'bootstrap/dist/css/bootstrap.min.css';
    import { Container } from "react-bootstrap";
    import NavBar from "./components/NavBar";
    import MoviesList from "./components/MoviesList";
    import axios from "axios";

    // 2] functional component section :
    function App() {
    // a] Define the useState variable :
    const [movies, setMovies] = useState([]) ;

    // b] Define a seperated function of recievcing a api variable and assinging it's value inside a useState variable :
    const getAllMovies = async () => {
    // 1- Define variable of recieving the datasource of api [using (axios) method wiht (async await) ] :
    const res = await axios.get('https://api.themoviedb.org/3/movie/popular?api_key=893adba5dca668458f2a181bcbd88e02&language=ar') ;

    // 2- printing testing of the defiend variable of api [accessing to the 'result' prop] :
    console.log(res.data.results);

    // 3- setting [variable of recieving the datasource of {api} as the value of defined [state] [accessing to the 'result' prop] :
    setMovies(res.data.results);
    }

    // c] Define a useEffect to implememnt the denfined sperated function => [to bve caled while the page first rendering] :
    useEffect(() => {
    getAllMovies() ;
    } , [])

    // d] Define a seperated function of searching proccessing: a api variable and assinging it's value inside a useState variable :
    const search = async (word) => {
    if (word ==='') {
    getAllMovies()
    } else {
    const res = await axios.get(`https://api.themoviedb.org/3/search/movie/popular?api_key=893adba5dca668458f2a181bcbd88e02&query=${word}&language=ar`);
    setMovies(res);
    }
    }

    // 3] the return section :
    return (
    < div className="color-body" >
    < NavBar / >
    < Container >
    // Sending the [state] variable of datasource as a prop into the component that using it :
    < MoviesList movies={movies} / >
    < /Container >
    </ div >
    );
    }


    // 3] export section :
    export default App;

2] {NavBar.js} Recieving the defined function into the NavBar component :

  • Recieving the functional paramter inside {NavBar} component
  • Define a parent function to use the functional paramter , with the paramter from which it will be used [input element]
  • Passing through parmamter into the recieved functional parameter {search} => to be assinged where it has been defined the main {App.js} component => to be used as the data Source of in the main page
function NavBar = ({search}) => {
// a] Define the parent searching container :
const onSearch = (word) => {
// passing through the parameter (word) [comming from the input value element] of the parent function into the functional parameter {search} :
search(word);
}

return (
< input
// applying the parent function within onChange event of the input , to recieve the input value as it's paremeter value
onChange={(e) => onSearch(e.target.value)}
type="text"
className="form-control"
placeholder="ابحث ..."
/ >
)
}


export default NavBar;
