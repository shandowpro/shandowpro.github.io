36 - useState concept :

useState() concept :

  1. It is one of builtin functions in the reactJs , which is used to store a variable's data to use it inside the functional component
  2. using this function need to :

    import the [useState] object from the main React library inside the file using it

    The defined [useState] variable must be inside the fuctional component , and before the return

    it is about array consists of (2) main elements , as following :

    1. [main varialbe] => which will store the data inside it
    2. [Handler function ] => which will used to change and control the main variable of the useState

  3. It is possible to use the useState hnadelr function directly or by using the conatainer function to access to the defined useState
  4. To import the {useState} object from the main react library => we must use the destructure curley brackets {}
  5. The (intial value) => It is the starting value of state , and can be any data type => [string ,number , array, object ]
  6. The defind {useState} varialbe => when changed => the whole functional compnoent will be => Re-Rendered




{useState()} Syntax :
// Importing [useState] object from the main reactJs [with the same line of React importing ] :
import {useState} from 'react' ;

// Define/extract the variable of the imported [useState] :
const [stateVar , stateVarHanlderFunction] = useState(initial Value) ;

// Define a container function to use the defiend [useState] inside it :
const containerStateHandler = () => {
// Calling the defined state handler function inside the container function :
stateVarHanlderFunction( codes to be implemented ) ;
}

// using/calling the [useState variable] inside it an Element inside the script :
< element > {stateVar} < /element >

// using/calling the [useState variable ] inside it an input Element's value property/attribute inside the script :
< input value={stateVar} />

// using/calling the [useState Handler function ] inside an Element's event , to apply the function codes when this event is occured :
< element onChange={stateVarHanlderFunction} > < /element >

{useState()} => usages and importance :

  1. It is critical to use in any compont in the react app [nearly in all conponents ]
  2. Some of common usages :

    To store a dynamic value [from the current client side or for the server] to use it inside the current functional component

    To use it's handler function with element event => to apply certain action or functionality inside the cuurnent functional component

    can be used with the element events such as [ onClick , onChange , onSubmit]
