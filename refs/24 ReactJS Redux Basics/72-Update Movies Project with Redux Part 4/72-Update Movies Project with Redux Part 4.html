72-Update Movies Project with Redux Part 4 concept :

Main Points in this lesson => define multiple reducers with [Redux Devtools] system as following steps :

A] Continuing the practical Applying {redux} system steps on using the {middleware} & {thunk} inside the project's files to solve the non compatiblity between the {redux} and {aync-await} on both Second funciton & Third function ] , as following :

  1. Converting [ {Second function}: [live searching] according to inserted value inside the input ] according to Redux system :
    1. 1] {movieAction.js} => Updating the built action method of Second Method {getMovieSearchs} of the application {getAllMovie} [getting all serached movies data from api accordig to the inserted value in the input] by using the new concept of redux [higher order function] the following :
      // (a) Importing section :
      // importing types expressions values to be used within the disptached action methods in this component:
      import { moviesType , moviesApi } from '../types/moviesType';
      import {axios} from 'axios' ;

      // (b) Define the main action method of getting all data of the movies from api [wih using parameter of return serach value ] :
      export const getMovieSearch = (word) => {
      // Using the higher order function method to make the redux accept the [async-await] :
      return async (dispatch) => {
      // [B : Second function] of [{live searching} according to inserted value inside the input] by using the redux system ] via => Define the variable of getting the api of the movies data , by using the imported variable of [url value] & [word] parameter :
      const res = await axios.get(https://api.themoviedb.org/3/search/movie?query=${word}&api_key=893adba5dca668458f2a181bcbd88e02);
      // Returning the action method's object data using the dispatch :
      dispatch({
      // 1- [type's expression value] - > as the imported variable of the movie type expression form {moviesType.js} :
      type: moviesType ,
      // 2- [data array of the serathed movies data to be obained form the assinged api ] -> by assinging extension [data.results] of the varaible of axios obtainted [search url api] value :
      data: res.data.results ,
      // 3- [page prop] -> by assinging extension [data.total_pages] of the varaible of axios obtainted [search url api] value :
      pages : res.data.total_pages
      })
      }
      }

    3. 3] Adjusting and updating {NavBar} component file of {NavBar.js} by adding all related codes of the Second function of [Dynamic Live search] & [Getting movies data accordng to inserted value of input] according to the using {middleware} & {redux-thunk} system , as following :
      • Define the {dynamic live searching} method , as following :
        1. a- The importing section , including :
          • { useDispatch } from "react-redux" => Requried to extract the disptach method to be used to send the action method into the main {Reducer} method
          • { getAllMovie , getMovieSearch } from "../redux/actions/movieAction"; => both are deifned actions methods in within the redux

        2. b- NO props needed to be sent inside this component

        3. c- Defined Variables and Functions => [inside the functional component and before the main return ] :
          • { const dispatch = useDispatch() } => the new Extracted dispatch method , to be able for dispatching the imported actions methods [getAllMovie , getMovieSearch]
          • { search(word) } => defined method of conditional executing for both [imoprted actions methods {getAllMovie , getMovieSearch} ] , with using {word} parameter to passs it through into the {getMovieSearch()} dispatched action method
          • { onSearch(word) } => parent container of defined method {search(word)}



      • Practical scripts :
        1. [NavBar.js] :
          // A] Importing section :
          // importing the [logo image] from the images folder for best pracitise => to be used in {NavBar} :
          import logo from '../images/logo.svg' ;

          import React from "react";
          import { Col, Container, Row } from "react-bootstrap";
          // importing the [useDisptach] class from [react-redux] to be sued in sendgin the recieved actions methods into the reducer and then store :
          import { useDispatch} from 'react-redux';
          // importing the [getMovieSearch] defined action method from [ movieAction.js] to be used to send the dispatch mehtod to reducer method :
          import { getAllMovie , getMovieSearch } from '../redux/actions/movieAction';
          // ------------------------------------------

          // Define the [B: Second function] : [dynamic live searching] method :
          const NavBar = () => {
          // 1] Define the disptach extracted method :
          const dispatch = useDispatch() ;

          // 2] Define the main method of the searching [according to the redux system ] :
          const search = async (word) => { if (word === "") {
          // Executing the first action method {getAllMovie} incase of inserted value of the input [if this value is empty] :
          dispatch(getAllMovie());
          } else {
          // Executing the second action method {getMovieSearch(word)} according to recived value of inserted value of the input [of this vallue is NOT empty] :
          dispatch(getMovieSearch(word));
          }
          };

          // 3] Define the parent searching container => to prevent the infite executive looping :
          const onSearch = (word) => {
          // passing through the parameter (word) [comming from the input value element] of the parent function into the functional parameter {search} :
          search(word);
          };
          // ------------------

          return (
          < div className="nav-style w-100" >
          < Container >
          < Row className="pt-2" >
          < Col xs="2" lg="1" >
          < a href='/' >
          < img className="logo" src={logo} alt="logo image" / >
          < / a >
          < /Col >
          < Col xs="10" lg="11" className="d-flex align-items-center" >
          < div className="search w-100" >
          < i className="fa fa-search"> </i >
          < input
          onChange={(e) => onSearch(e.target.value)}
          type="text"
          className="form-control"
          className="form-control"placeholder="ابحث ..." />
          < /div >
          < /Col >
          < /Row >
          < /Container >
          < /div >
          );
          };


          export default NavBar;
