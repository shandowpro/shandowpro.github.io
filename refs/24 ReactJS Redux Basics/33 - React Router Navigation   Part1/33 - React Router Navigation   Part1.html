33 - React Router Navigation Part1 concept :

Navigation Router in the ReactJs => concept :

  1. It is about a method for navigation between defined pages and components
  2. Every page in the react is called a [router]
  3. The diffenreces of Navigation in both and {reactJS} & {default Html} in concept process :
    • Concept =>
      1. {default Html} => Depends on reloading the target location url tio access to the url
      2. {reactJS} => Depends on {SPA} to access the to the target url , without reloading the current page
    • Mechanism =>
      1. {default Html} =>

        Send the target location url [page router] to server to and then reload ther current page

        use the assigned url value in the current request


      2. {reactJS} =>

        Render the curent page router with new rquested page router , without relaoding the current page

        use the assigned url value in certain library of list of routers in the current page , assgined in the [App.js] inside special element





Navigation Router in the ReactJs => Requriments :

  1. setup and install the {react-router-dom} in the current project main path , usintg the folloiwng codes :
    npm i react-router-dom ;
  2. {react-router-dom} properties as following :
    • must be imported in the main [App.js] as following overall importing code :
      import {BrowsRouter , Routers , Router} from 'react-router-dom' ;
    • each imported object represent a certain function :
      1. {BrowserRouter} => represent the main container of all defined pages routers in the main project
      2. {Routers} => represent the inner conatiner of all defined routers
      3. {Router} => represent the single page router , and incoluds the follwing properties :
        • [path] => will include page router's path
        • [element] => will include page router's Element Tag




Navigation Router in the ReactJs Syntax => [App.js] ::
// Importing the required libraries for routering [react-router-dom ] :
import {BrowerRouter, Route , Routes} from 'react-router-dom' ;
// Importing the defined and used components [to be called as pages routers || fixed components ] :
import fixedComp from './component/fixedComp' ;
import routerComp1 from './component/routerComp1' ;
import routerComp2 from './component/routerComp2' ;
import routerComp3 from './component/routerComp3' ;
const App = () => {
return (
// Calling the fixed component :
< fixedComp >

// start using the Pages Routers to assign defined components as page Routers :
< BorwerRouter >
< Routers >
// define the first router of component [routerComp1] :
< Router path='/routercomp1' element={< routerComp1 /> } />
// define the second router of component [routerComp2] :
< Router path='/routercomp2' element={< routerComp2 /> } />
// define the third router of component [routerComp3] :
< Router path='/routercomp3' element={< routerComp3 /> } />
< /Routers >
< /BorwerRouter >

// Calling the Footer as fixed component :
< Footer >
)

}




Navigation Router in the ReactJs => Rules ot use :

  1. Install and setup: {react-router-dom} in main project path

  2. import the main objects from : {react-router-dom} in the main page {App.js}

  3. Define each component {as Route} element slefclosed => inside the main continer of {BrowserRourter} and inside the inner container {Routers}

    4. /
  4. each [Route] must include the following [2] properties :
    1. {path} => stirng value of page browser path , to be used ot naviagate to this page router => (must be in small case)
    2. {element} => include the page/component's Tag inside {} => [to be rendered by reactJS]

Navigation Router in the ReactJs => practical examples =>
// Importing the required libraries for routering [react-router-dom ] :
import {BrowerRouter, Route , Routes } from 'react-router-dom' ;
// Improting the defined components to be called as pages routers :
import About from './component/About' ;
import Content from './component/Content' ;

// Improting the defined components to be called as fixed components :
import NavBar from './component/NavBar' ;
import Footer from './component/Footer' ;

const App = () => {
return (
// Calling the Navbar as fixed component :
< NavBar >

// start using the Pages Routers to assign defined components as page Routers :
< BorwerRouter >
< Routers >
// define the router of the about page :
< Router path='/about' element={< About /> } />
// define the router of the content page :
< Router path='/content' element={< Content /> } />
< /Routers >
< /BorwerRouter >

// Calling the Footer as fixed component :
< Footer >
)

}

Important Notes :

  • Define the {Routers} in the main [App.js] :
    1. Must import the {react-router-dom} with all it's objects {BrowerRouter, Route , Routes } to use it properly in the defining the routes
    2. Inside each [Route] => {path} , must write the path of each router in small letter = > [to be read properly by the brwoser]
    3. Inside each [Route] => {element} , must write the router page as an component self colsed Tag = > [to be read properly by the ReactJs script]
    4. We must add only the page routers , not the fixed pages/components
